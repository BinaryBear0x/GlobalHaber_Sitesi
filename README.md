# GlobalHaber_Sitesi

GlobalHaber_Sitesi, Flask tabanlı bir haber sitesi denemesi olarak hazırlanmıştır. Proje; haber toplama, listeleme ve detay sayfası mantığını bir araya getirir.

## Ne Yapar

- haberleri veritabanından listeler
- sayfalama ile içerik gösterir
- detay sayfasında içerik ve kelime sayımı sunar
- scraping ve çeviri yardımcı dosyaları içerir

## Teknolojiler

- Python
- Flask
- HTML
- CSS
- SQLite / SQLAlchemy tabanlı model yapısı
